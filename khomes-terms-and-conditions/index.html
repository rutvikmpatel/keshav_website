2. Description of Service

kHomes is a visitor management application designed for residential complexes and gated communities. The App provides the following services:

  • Visitor Entry Management: Digital visitor registration and tracking
  • Visitor Notifications: Real-time notifications to residents about visitor arrivals
  • Visitor Approval System: Resident approval/rejection of visitor requests
  • Visitor Image Capture: Photo documentation of visitors (upcoming feature)
  • Pre-approved Pass System: QR code-based visitor passes (upcoming feature)

3. User Types and Registration

3.1 User Categories

  • Residents: Homeowners/tenants living in the society
  • Security Guards: Personnel responsible for visitor management
  • Visitors: Individuals visiting residents
  • Admins: Society administrators with management privileges
  • Chairman: Society chairman with highest level permissions

4.2 Resident Permissions

  • Approve or reject visitor requests

4.3 Security Guard Permissions

  • Enter visitor details
  • Select home numbers for visitor requests
  • Receive approval/rejection notifications
  • Scan QR codes for pre-approved passes (upcoming feature)

5. Approval Workflow

The visitor approval process works as follows:

  1. Security guard enters visitor details and selects home number
  2. Request is sent to all residents of that home who have opted for notifications
  3. Any resident can approve or reject the request
  4. Security guard is notified of the decision
  5. If no resident responds, the request remains pending
